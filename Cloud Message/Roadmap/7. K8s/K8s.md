Mục tiêu: Đưa toàn hệ thống lên Kubernetes.

**Học:**

- Dockerfile + Docker Compose
    
- Kubernetes cơ bản (Pod, Deployment, Service, Ingress)
    
- Helm chart
    
- Redis + Kafka deployment trên K8s (dùng Bitnami chart)
    

**Làm:**

- Deploy Backend API, Broker, Kafka, Redis, MariaDB
    
- ConfigMap, Secret cho connection string
    
- Setup ingress route cho REST API và broker
    

**Sản phẩm:**

- Toàn hệ thống chạy trên K8s cluster local (minikube / k3d / kind).
