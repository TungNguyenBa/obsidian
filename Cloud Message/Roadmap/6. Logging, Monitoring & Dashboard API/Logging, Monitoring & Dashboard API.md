Mục tiêu: Giống Firebase console – có logging & analytics.

**Học:**

- Serilog + Seq hoặc Elasticsearch
    
- OpenTelemetry tracing
    
- Prometheus + Grafana basics
    

**Làm:**

- Log mọi event: send, deliver, read
    
- API: `/logs`, `/analytics/messages`
    
- Dashboard backend: thống kê message volume theo ngày, tỉ lệ delivery, v.v.
    

**Sản phẩm:**

- API có thể show thống kê message giống Firebase Console backend.
